Case Study: ContentMind

The Challenge

Build a fully autonomous content production system that detects trends, writes scripts, generates videos with AI narration, quality-scores content through a self-improving council, and publishes to YouTube daily — all without human intervention and at zero monthly cost.

The Engineering Approach

ContentMind is a 7-layer LangGraph pipeline orchestrating 14 specialised agents across trend detection, RAG-grounded script writing, multi-voice TTS, AI image generation, quality evaluation, and autonomous publishing. The system demonstrates production-grade engineering across every layer:

  • Strategic Prompt Engineering

    Reduced the Quality Council from 5 parallel LLM calls to a single comprehensive evaluation — cutting Groq API usage by 60% while maintaining identical scoring accuracy. The 11-technique storytelling system prompt achieves 75%+ council scores consistently without human editing.

  • Token Optimization

    Full production deployment completed in under 100 AI assistant credits. Every prompt is precision-engineered: 70-80 word scripts (30-second videos), structured JSON responses, escalating improvement loops that get more aggressive per retry. Zero wasted tokens.

  • Cost Engineering (₹0/month)

    Architected the entire production system to run on free-tier cloud services through strategic vendor selection and resource optimization. No paid subscriptions required for daily autonomous operation — demonstrating that production-grade AI systems don't require expensive infrastructure when designed correctly.

  • Self-Improving Quality System

    The pipeline never fails. A true score→improve→rescore loop with escalating thresholds (75→75→65) guarantees publishable content. Past performance data from Qdrant calibrates future scoring — the system gets smarter with every video.

  • QA-Driven Architecture

    Applied QA engineering principles to AI: semantic deduplication (prevents repeat content), niche relevance filtering (rejects off-topic trends), and multi-layer graceful fallback chains ensuring the pipeline never fails — even when individual services are unavailable.
